## 安全最佳实践
1. 服务端私钥必须离线存储，不得暴露在公网环境
2. 用户授权确认必须使用多因素认证（MFA）
3. 所有握手请求必须进行速率限制，防止暴力攻击
4. 定期审计授权日志，发现异常行为及时处理
