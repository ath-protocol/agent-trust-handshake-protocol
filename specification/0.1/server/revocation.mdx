## Token Revocation

**`POST /ath/revoke`**

Revokes an ATH access token, making it immediately unusable.

## Request

```json
{
  "client_id": "ath_travelbot_001",
  "token": "ath_tk_xxxxxxxx"
}
```

### Request Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `client_id` | string | Yes | The agent's client ID |
| `token` | string | Yes | The ATH access token to revoke |

## Response

Returns `200 OK` on successful revocation.

## Behavior

- The token becomes immediately unusable after revocation
- Subsequent API calls using the revoked token MUST be rejected
- The agent, the user (through a management UI), or the administrator can revoke tokens
- Revocation SHOULD be logged for audit purposes
