# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in the ATH protocol specification or documentation, please report it responsibly.

**Do NOT open a public GitHub issue for security vulnerabilities.**

Instead, please email the maintainers or use [GitHub's private vulnerability reporting](https://github.com/ath-protocol/agent-trust-handshake-protocol/security/advisories/new).

## Scope

This security policy covers:

- The ATH protocol specification
- The documentation website content
- The OpenAPI schema

For security issues in specific implementations (SDK, gateway), please report to the relevant repository.

## Response

We will acknowledge receipt within 48 hours and provide a timeline for a fix.
